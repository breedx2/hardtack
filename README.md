A generic Teensy-based eurorack module.

It's very early in the development/idea phase.

Plans:

* teensy
* small cap touch tft
* audio codec - 2 ins, 2 outs
* 60s of audio memory
* peripherals: 2 rotary encoders, 2 buttons, 2 switches
